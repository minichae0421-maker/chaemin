# 프로젝트 기본 설정

## 언어 설정
- 모든 응답은 **한국어**로 작성
- 코드 주석도 한국어로 작성
- 에러 메시지 설명도 한국어로

## 프로젝트 구조
```
my_ai_wep/
├── .claude/                    # Claude 설정
├── lecture1/                   # 실습 프로젝트
│   ├── docs/                   # 문서
│   ├── _template_settings/     # 템플릿 설정
│   └── my-react-app/           # React 앱
├── index.html                  # 메인 랜딩 페이지
└── CLAUDE.md                   # 이 파일
```

## 기본 원칙
1. 모든 코드 설명은 초보자도 이해할 수 있게
2. 단계별로 차근차근 설명
3. 실습 위주의 학습 지원
