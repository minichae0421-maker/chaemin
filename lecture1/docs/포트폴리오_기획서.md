**작성일**: 2025년 1월 22일
**목표**: 웹디자이너 취업을 위한 차별화된 포트폴리오

---

## 1. 포지셔닝 전략

### 타겟 프로필
| 항목 | 내용 |
|------|------|
| 목표 직무 | 웹디자이너 |
| 경력 수준 | 신입 / 취준생 |
| 지원 회사 | 스타트업, 에이전시, 대기업 |
| 핵심 강점 | 디자인 감각 + 소통 능력 |

### 차별화 포인트
- 디자인과 개발 모두 가능한 "풀스택 디자이너"
- AI에 대한 관심과 트렌드 이해
- 네이비+노랑의 강렬한 브랜드 컬러

---

## 2. 디자인 시스템

### 컬러 팔레트
| 색상 | 코드 | 용도 |
|------|------|------|
| 네이비 | #1A1A5E | 메인 컬러, 텍스트 |
| 노랑 | #FFF200 | 강조, 포인트 |
| 오프화이트 | #F8F8F6 | 배경 |
| 화이트 | #FFFFFF | 카드, 콘텐츠 영역 |

### 폰트 스타일
- **헤드라인**: 볼드, 임팩트 있는 스타일
- **본문**: 깔끔하고 가독성 좋은 스타일
- **특수 요소**: 메뉴판 스킬 섹션에 이탤릭체 활용

### 레이아웃
- 미니멀 + 카드형 조합
- 충분한 여백으로 시원한 느낌
- 섹션별 배경색 변화로 구분

## 3. 섹션별 상세 기획

### Hero 섹션 (임팩트형)
```
┌─────────────────────────────────────┐
│                                     │
│         안녕하세요,                  │
│    웹디자이너 이채민입니다            │
│                                     │
│   "즐겁게 디자인하고,                │
│    함께 성장합니다"                  │
│                                     │
│        [ 포트폴리오 보기 ]            │
│                                     │
└─────────────────────────────────────┘
```

**구현 요소**:
- 큰 타이포그래피로 이름과 직함 강조
- 슬로건/가치관 표현
- CTA 버튼으로 Projects로 유도
- 미니멀하지만 임팩트 있는 레이아웃

---

### About Me 섹션 (스토리텔링)

**구현 요소**:

- 개인적 매력: AI 관심사
- 성장 목표: 시니어 프론트 개발자/웹 디자이너

```
│                                     │
│  HTML ...................... *상*   │
│  CSS ....................... *상*   │
```

- 숙련도는 *이탤릭체*로 표시
- 점선으로 연결 (클래식한 메뉴판 느낌)
